Инструкция

Как использовать FunMapApp:

  1. Используйте поиск для нахождения интересных мест
  2. Фильтруйте места по категориям и популярности
  3. Нажимайте на маркеры для получения подробной информации
  4. Сохраняйте понравившиеся места в избранное
  5. Строите маршруты до выбранных мест
  6. Делитесь местами с друзьями
  7. Используйте AR-режим для навигации
  8. Создавайте собственные маркеры на карте (правой кнопкой мыши)
  9. Просматривайте информацию о пробках и дорожной ситуации
  10. Переключайтесь между различными типами карт
  11. Используйте режим для слабовидящих в настройках
  12. Включайте оффлайн-режим при плохом соединении
  13. Просматривайте историю своих перемещений
  14. Совместно редактируйте маршруты с друзьями
  15. Анализируйте свою активность в разделе аналитики

AR-режим навигации

Как использовать AR-режим:

  1. Держите устройство вертикально перед собой
  2. Медленно поворачивайтесь, пока не увидите маршрут
  3. Следуйте указаниям на экране
  4. Для лучшего результата используйте заднюю камеру

Маршрут будет отображаться поверх реального мира с помощью AR-технологий.
